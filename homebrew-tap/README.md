# homebrew-tap

Homebrew tap for ssrok - Blazing fast secure reverse proxy tunnel.

## Installation

```bash
brew tap selcuksarikoz/ssrok
brew install ssrok
```

## Available Formulae

- `ssrok` - Client binary

## Updating

```bash
brew update
brew upgrade ssrok
```

## Uninstalling

```bash
brew uninstall ssrok
brew untap selcuksarikoz/ssrok
```
